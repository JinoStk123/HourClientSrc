# Open Hour

![Preview](/images/image2.png)

[Hour Client](https://myau.sell.app/), for those who can’t afford it.

Based on **`Hour-250910`**, with added features and improvements not found in the original, focused on expanding and refining the original Hour client.

[![Discord](https://invidget.switchblade.xyz/yjSaTufmqU)](https://discord.gg/yjSaTufmqU)

## Changes

* Added Click GUI (ported from Raven B3 by blowsy)
* Fixed bugs

If you encounter an issue or have a feature suggestion, please [create an issue](https://github.com/60124808866/OpenHour/issues).

## Building

```bash
./gradlew build
```
